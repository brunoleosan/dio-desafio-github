comandos gitbash
cd - abre a pasta selecionada
git status - mostra o status do repositório
git add . ou -A - adiciona todas as pastas novas ao repositório no Github 
git commit -m "" - adiciona uma mensagem/comentário ao commit
git push origin main - envia o commit para o Github
